# SNAKE
A klasszikus **snake** játék extra beálításokkal és funkciókkal.
### Funkciók:
- Különböző méretű pályák
- Különböző gyorsaságú kígyók
- Almák mennyisége
- Almák kinézete
- A kígyó egyéb szerekesztése (szín, szemek)
